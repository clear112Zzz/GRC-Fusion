# GRC-Fusion: Geometric Radar-Guided Feature Modulation and Asymmetric Temporal Fusion for 3D Object Detection

This is the official repository for the paper **"GRC-Fusion: Geometric Radar-Guided Feature Modulation and Asymmetric Temporal Fusion for 3D Object Detection"**.

**News & Status**
* **[Current Status]**: The source code is currently undergoing final cleanup and institutional review. We are actively preparing the codebase to ensure reproducibility and readability. 
* The full code, including data preparation, training scripts, evaluation scripts, and pretrained weights, will be made publicly available immediately upon the acceptance of the paper. Please stay tuned!

**Highlights & Performance**
GRC-Fusion proposes a robust radar-camera fusion framework to address cross-modal spatial misalignment and feature pollution. Evaluated on the **nuScenes** dataset, our method achieves state-of-the-art performance:

* **Validation Set**: 49.6% mAP | 58.5% NDS
* **Test Set**: 59.5% mAP | 65.8% NDS
